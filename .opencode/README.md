# OpenCode Compatibility Surface

OpenCode should use the same shared workflow surface as Codex in this repo.

## Use These Files

- `AGENTS.md` for always-on workspace instructions
- `OPENCODE.md` for OpenCode-specific compatibility notes
- `.codex/prompts/` for reusable prompt docs
- `.codex/skills/` for reusable skill docs
- `.codex/README.md` and `.codex/prompts/README.md` for navigation

## Rule

Do not create a parallel OpenCode-only prompt or skill tree unless the shared `.codex/` surface becomes insufficient. The default is one shared cross-tool workflow surface.
