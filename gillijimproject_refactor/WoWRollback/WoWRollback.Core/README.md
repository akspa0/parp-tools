# WoWRollback.Core

## Overview
Shared utilities and models used across the WoWRollback toolkit.
- File helpers, path resolution, logging
- Common data models and result types
- Reusable services (archive access, DBC helpers)

## Usage
- Add a `ProjectReference` to `WoWRollback.Core` from your tool or module.
- See usages in `WoWRollback.Cli` and `WoWRollback.Orchestrator` for patterns.
