Test Sedimentary Layers

Open browser console (F12) and run these commands to test:

// Check if sedimentaryLayers exists
console.log('sedimentaryLayers:', window.sedimentaryLayers);

// Check registered markers count
if (window.sedimentaryLayers) {
    console.log('Registered UIDs:', window.sedimentaryLayers.uniqueIdToMarkers.size);
    console.log('Total markers:', Array.from(window.sedimentaryLayers.uniqueIdToMarkers.values()).reduce((sum, arr) => sum + arr.length, 0));
}

// Check ranges loaded
if (window.sedimentaryLayers) {
    console.log('Ranges loaded:', window.sedimentaryLayers.ranges.length);
    console.log('First range:', window.sedimentaryLayers.ranges[0]);
}

// Manually trigger filter
if (window.sedimentaryLayers) {
    window.sedimentaryLayers.applyFilters();
}

Expected Output:

  • sedimentaryLayers: should be an object, not undefined
  • Registered UIDs: should be > 0 after markers load
  • Ranges loaded: should match number of ranges in CSV after clicking "Load UniqueID Ranges"
  • applyFilters(): should show filter logs in console
